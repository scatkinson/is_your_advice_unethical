About

The model for this app was trained using data from Reddit. Half of the data come from the r/LifeProTips subreddit, and half come from r/UnethicalLifeProTips subreddit. You can read about the project in the corresponding blog post found here. The "Unethical Score" returned is a quantification of how strongly the model assesses your advice input to be unethical--the higher the score, the more unethical; anything higher than 50% is determined to be unethical by the model.

This model was deployed using the Flask API in Python via the Heroku Cloud Application Platform. You can find all relevant code including the Jupyter notebook at the corresponding GitHub repository. The HTML for this site was found at this repository which corresponds with this helpful Flask tutorial video.
